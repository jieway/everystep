# 第一章

面试流程：电话面试/远程面试 => 现场面试

* 首先明确题意，思考清楚后再写代码，明确题意后最好先造数据，然后写完代码再进行单元测试。
* 代码出现问题后调试，设置断点，单步跟踪，查看内存，分析调用栈。快速发现问题根源并解决问题。
* 注意要有良好编码习惯，例如良好的代码命名，缩进对齐。
