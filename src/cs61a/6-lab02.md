## Q4: Count van Count

首先返回值是一个函数，所以需要在内部重新定义一个函数用于返回。

```python
def count_cond(condition):
    def count(n):
        i, count = 1, 0
        while i <= n:
            if condition(n, i):
                count += 1
            i += 1
        return count
    return count
```

## Q8: Composite Identity Function

```python
def composite_identity(f, g):
    return lambda x : f(g(x)) == g(f(x))
```

## Q9: I Heard You Liked Functions...

```python
def cycle(f1, f2, f3):
    def ret_fn(n):
        def a(x):
            i = 0
            while (i < n):
                if (i % 3 == 0):
                    x = f1(x)
                elif (i % 3 == 1):
                    x = f2(x)
                elif (i % 3 == 2):
                    x = f3(x)
                i += 1
            return x
        return a
    return ret_fn 
```
